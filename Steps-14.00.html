Adding Project Title, Refreshed Date and Date Slicer 14


Steps

Refreshed Date

Open Visualization Pane

Select Visual Card

Drag the visual to the dashboard.

Click on refreshed date from Data Refresh Table and place it inside the fields.

General: Properties, Height = 83, Width =221, Position Horizontal = 223, Vertical = 0

Visual: Callout Value, Color = #000000, Font Size = 12, Font type = Segoe UI

Category = OFF

General: Title ON, Text Title = " Refreshed Date", Font Color = #000000, Background Color = #FFFFFF,   Font Size = 12  and Font Family = Segoe U

Steps

Adding Project Title 

1.     Click on the Text box from Home or insert ribbon.

2.     Begin typing the name of the project [(World Products Sales (2010-2017)]

3.     Select the text then change the font size to 24 and font type to Segoe UI

4.     Select the text then Alignment to center.

5.     General:  Properties, Height = 83 and Width = 836 , Position, Horizontal =444, Vertical = 0

6.     Select Effects then turn off the background to =0

Visual border = on, color=#D3D3D3


Steps

Adding Date Slicer

1.    Open Visualization pane

2.    Select Slicer visual

3.    Drag the visual to the dashboard.

4.    Click on Date table and drag Year column from the Date Hierarchy then place it inside the field.

5.    General: Properties, Height = 112, and Width = 231, Position, Horizontal = 0, Vertical =83

6.     Slider header = On, Slider On, Color = #605E5C, Values color = #605E5C, font type Segoe Arial, Size = 10, Background Color = #E6E6E6, General title On, enter text " Year", font type Arial, size=18, text Color = #000000, Background = #A3CBF1

Effect, Background =on, Color=F9F6F6

Visual Border = On, color=D3D3D3
