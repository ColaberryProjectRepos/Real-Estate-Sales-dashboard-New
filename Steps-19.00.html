Creating Scatter Chart-Residential Type 19


1.Steps to build - Scatter Plot Chart

From the visualization select Scatter Plot Chart

Fields: 

Ø Values: residential type

Ø X axis value: sales per MTh

Ø Y-axis value: sales YOY%

Ø Tool tips: sales MOM%, sales YOY %, property tax%

 

Formatting the Visual:

Ø General:  properties   size; Height = 311; width= 526; position horizontal = 228; vertical = 409

Ø Title - ON; Title Text – Residential Type; Title Heading - Normal; Font Color - #000000; Background Color - #A0D1FF, Alignment - middle; Text Size - 18; Font – Arial

Ø Bold

Ø Effect=ON, Background Color=F9F6F6, Visual Border= On, Color=D3D3D3

Ø Tool tips, Text Font= Arial, size= 10, Label color=#000000, Bold, Values Color =#605E5C, Background color=#FFFFFFF

 

Ø Legend:  ON; Position- Left; Color- #605E5C; Font – Arial, Bold;                          Text Size – 10; Title Text Sales MOM%

Ø X-Axis - ON; Scale Type - Linear; Start -Auto; End -Auto; Color - #605E5C; Text Size – 10.

§  Font – Arial, Bold; Title - ON; Title Color - #000000; Title size - 12.

Ø Y-Axis (Column)- ON; Position-Left; Scale type - Linear; Start & End - Auto; Color -color - #605E5C; Text Size- 10.

Ø Font – Arial, Bold; Title - ON; Title Color - #000000; Title size 12.                                  

Ø Markers: Type =Circle(.), Format style= Gradient, Filed =Sales MOM%, Minimum values color=#00008B, maximum values color=#0D02F9.

Ø Category Label = On, Values Font=Arial, Size=10, Bold, Color=#605E5C.

Ø Legend=On, Top Left, Text Font= Arial, Bold, size=11, Color=#605E5C, Title Text= Sales MOM%.
